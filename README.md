# 📊 Student Academic Performance Dashboard (Excel Project)

## 📌 Project Overview

Student Academic Performance Dashboard built using Microsoft Excel with automated marks calculation, grade assignment, pass/fail analysis, and data visualization.

## ✨ Features

- Student Marks Management
- Automatic Total Marks Calculation
- Grade Assignment
- Pass/Fail Status Detection
- Subject-wise Performance Analysis
- Highlighted Total Marks
- Bar Chart Visualization
- Academic Performance Dashboard

## 📋 Table Structure

| S.No | Student Name | Student ID | English | Maths | Physics | Total Marks | Grade | Status |
|------|-------------|------------|---------|-------|---------|------------|-------|--------|
| 1 | Student 1 | ST001 | 85 | 90 | 78 | 253 | A | Pass |
| 2 | Student 2 | ST002 | 35 | 40 | 28 | 103 | Fail | Fail |
